Acerca de este proyecto

¿Qué es esta página?

Es una pequeña página web educativa creada con Flask. Muestra secciones con imágenes y texto sobre autos antiguos, animales exóticos y las siete maravillas del mundo.

¿Para qué sirve?

Sirve como ejemplo práctico para aprender cómo organizar y mostrar contenido en una página: navegar con un menú, ver imágenes y leer información simple sobre cada tema.

¿Cómo está organizada?

  • Página principal: resumen y acceso a las secciones.
  • Secciones: Autos Antiguos, Animales Exóticos, Las 7 Maravillas y un formulario de inicio de sesión (solo de ejemplo).

¿Cómo la uso? (pasos simples)

  1. Abre la terminal integrada en VS Code (o PowerShell) y cambia a la carpeta del proyecto:
    cd "c:\Users\HP`\Desktop\15-10-2025"
  2. Ejecuta la aplicación:
    python app.py
  3. Abre tu navegador y escribe http://127.0.0.1:5000. Usa el menú superior para moverte entre las secciones.
